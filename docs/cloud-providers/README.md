# Cloud Provider Integration Guides

This directory contains integration guides for various cloud and virtualization platforms.

## Supported Providers

### Public Cloud Platforms

- **[Alibaba Cloud (Aliyun)](alibabacloud-integration.md)** - Integration with Alibaba Cloud services
- **[AWS (Amazon Web Services)](aws-migration-guide.md)** - Migration guide for AWS environments
- **[Oracle Cloud Infrastructure (OCI)](oci-integration.md)** - OCI integration and setup

### Private Cloud & Virtualization

- **[OpenStack](openstack-integration.md)** - OpenStack cloud platform integration
- **[Proxmox VE](proxmox-integration.md)** - Proxmox Virtual Environment integration

## Overview

These guides provide detailed instructions for:
- Configuring HyperSDK to work with each platform
- Authentication and credential management
- Platform-specific migration workflows
- Best practices and limitations
- Troubleshooting common issues

## General Requirements

Before using any cloud provider integration:
1. Review the [Getting Started Guide](../getting-started.md)
2. Configure credentials as described in [Security Best Practices](../security-best-practices.md)
3. Ensure network connectivity to the target platform
4. Review platform-specific prerequisites in the individual guides
