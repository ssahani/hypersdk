# HyperSDK Monitoring Stack

Complete monitoring solution for HyperSDK using Prometheus, Grafana, and Alertmanager.

## Quick Start

```bash
cd monitoring
docker-compose up -d
```

Access:
- Grafana: http://localhost:3000 (admin/admin)
- Prometheus: http://localhost:9090
- Alertmanager: http://localhost:9093

## Documentation

See full documentation in DEPLOYMENT.md and online at https://docs.hypersdk.io/monitoring
